

## Contact Page

Intro: This page displays the main picture of the masjid, an iframe for the masjid's location via Google Maps, and basic contact details (WhatsApp, phone, email) set by the admin. It features a modern layout with a short description suitable for the project. The page includes a header and footer for consistency with the site's design.

1. Header and Footer: Add header and footer as described in HomePage.

2. Main Picture: Display a prominent picture of the masjid, set by the admin.

3. Location Iframe: Embed a Google Maps iframe showing the masjid's location.

4. Contact Details: On the left or remaining space, list basic contact information including WhatsApp, phone, and email, all configurable by the admin.

5. Description: Include a short description that suits the project, explaining the contact purpose or masjid details.
