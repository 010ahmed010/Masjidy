

## Absence and Attendance Page

Intro: This page serves as a comprehensive history viewer for student absence and attendance records. Visitors (parents/guardians) can check attendance for previous days, select a specific class, and view whether their children were present or absent on particular dates. The page includes a header and footer for consistency with the site's design.

1. Header and Footer: Add header and footer as described in HomePage (reusable component).

2. Class Selection: Provide a dropdown or list to select the student's class. This filters the attendance data to show only relevant records.

3. Date Selection: Allow users to choose a specific date or view a calendar/range of dates. Display attendance status (present/absent) for each student in the selected class on that day.

4. Attendance Display: Show a table or list of students with their attendance status. Include columns for student name, status (present/absent), and any notes (e.g., excused absence). Data is read automatically from the database set by teachers.

5. Search/Filter Options: Add search functionality to find a specific student or filter by attendance status.
